## 👤 About

Self-taught IT developer from **Knokke-Heist, Belgium**. I've been building software since ~2010 across Apple platforms, web, backend, data, and business analysis — no CS degree, no formal training, just learning by doing and shipping things that work.

I run **Reviewell BV**, build macOS and iOS apps, analyse business problems, and constantly add new tools to my stack.

**Dutch** (native) · **French** (fluent) · **English** (professional)
